标签

  • head 头部

放置文档的标题,<base>,<link>,<meta>等内容

  • 标签

注释，如python中的#

可以用来写注释，也可用来写一些页面暂时无法实现的脚本

  • div 标签

用来分节或分块 其实是无意义的标签

知道了div就知道了80%的内容

  • a 标签

中带有的属性 href 一般指向网站中的子页面 常用来用做翻页

  • input 标签

主要搜集用户数据，可以是文本，可以是文件

那么搜集的信息被提交到哪里了呢，被提交到了 form 标签指定的位置

  • form 标签

input 的内容被提交到form 中指定的链接🔗中

  • test 中的emjoi

emjoi 也是文字，同文字一样编写

  • button 标签

可以实现一些操作，如点击触动事件

属性

  • class 属性

加入样式，改变html外观，以及页面的布局,颜色，修改页面的大小

  • id 属性

主要用来识别，可以每一个都设置一个id，但用不到就没必要

  • style 属性

元素样式 也叫内联样式，一般不需要动

  • 事件

onclick 点击时执行后面的东西

可能会写在JS里面

有的框架可能就在上面

  • 指令

ic-action 就是一种指令

div 应用于大部分场景
